## How to install

```bash
git clone https://github.com/sergey0226/React_Redux_Sagas.git
cd react-redux-crud
```

In terminal:

```bash
yarn install
```

## How to run

In terminal:

```bash
yarn start
```

You can now view the client SPA in the browser.

    Local:            http://localhost:3000/
